# Dublin Air Pollution Prediction Using Machine Learning

## Project Overview
This project predicts street-level air pollution in Dublin City using Google Project Air View data. Machine learning models were used to identify pollution patterns and important predictors.

## Dataset
The dataset used is Google Air View Data – Dublin City, covering May 2021 to August 2022.

## Objectives
- Clean and prepare the air pollution dataset
- Extract spatial features from road geometry
- Build Random Forest and Gradient Boosting models
- Evaluate models using RMSE, MAE and R²
- Visualize pollution hotspots across Dublin

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Shapely
- Folium
- Matplotlib

## Models Used
- Random Forest Regressor
- Gradient Boosting Regressor

## Evaluation Metrics
- RMSE
- MAE
- R² Score

## Key Results
The models were able to predict street-level pollution patterns and identify high-pollution areas across Dublin.
